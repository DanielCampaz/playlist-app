# Playlist App (Api)

[Download and Install GO](https://go.dev/doc/install)

### Command to start the api

- go run main.go

### Authorization

In the header, the token must be sent for all endpoints except login and singup,
Example:

```json
{
  "Content-Type": "application/json",
  "Token": "Your Token"
}
```
